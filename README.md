## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/eshwar-21/Eshwar-portfolio.git
```

2. Navigate to the project directory:
```bash
cd Eshwar-portfolio
```

3. Install dependencies:
```bash
npm install
```

### Running Locally

To run the portfolio locally:

```bash
npm start
```

The portfolio will open in your browser at `http://localhost:3000`

## Building for Production

To create a production build:

```bash
npm run build
```

## Deployment

This portfolio is deployed to GitHub Pages. To deploy your own:

1. Build the project
2. Push to the `gh-pages` branch or configure GitHub Pages settings
3. Visit your portfolio at `https://yourusername.github.io/Eshwar-portfolio`
